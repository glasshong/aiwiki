Vision¶

Vision은 이미지를 입력으로 받아 분류, 객체 검출, 영역 분할 등을 수행하는 DL 분야입니다.

  • Convolution / Padding / Pooling
  • Image Classification

  • Swin Transformer
  • DETR

큰 흐름¶

CNN은 지역 패턴을 잘 찾고, ViT 계열은 이미지를 patch token으로 바꿔 전역 관계를 학습합니다.
