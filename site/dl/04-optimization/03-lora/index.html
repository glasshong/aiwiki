LoRA / QLoRA¶

LoRA는 큰 가중치 행렬을 직접 수정하지 않고 작은 저랭크 행렬을 추가해 학습하는 방법입니다.

LoRA¶

전체 모델을 업데이트하지 않고 작은 adapter만 학습합니다.

QLoRA¶

기반 모델을 양자화한 상태에서 LoRA를 적용해 GPU 메모리 요구량을 낮춥니다.

해결하는 문제¶

Full fine-tuning은 GPU 메모리와 저장 공간이 많이 필요합니다. LoRA/QLoRA는 학습 파라미터와 메모리 사용량을 크게 줄입니다.
