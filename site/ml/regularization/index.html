L1 / L2 정규화¶

정규화는 모델이 학습 데이터에 과하게 맞춰지는 것을 줄이기 위해 가중치 크기에 패널티를 주는 방법입니다.

L1¶

  • 가중치 절댓값의 합에 패널티를 줍니다.
  • 일부 가중치를 0에 가깝게 만들 수 있습니다.
  • 특징 선택 효과가 있습니다.

L2¶

  • 가중치 제곱합에 패널티를 줍니다.
  • 큰 가중치를 부드럽게 억제합니다.
  • Ridge regression, weight decay와 연결됩니다.

비교¶

구분 L1 L2
효과 희소성 안정화
가중치 일부가 0에 가까워짐 전체적으로 작아짐
사용 느낌 특징 선택 일반적 과적합 완화
