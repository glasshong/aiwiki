지도 / 비지도 / 강화 학습¶

학습 유형은 모델이 어떤 신호를 기준으로 배우는지에 따라 나뉩니다.

유형 학습 신호 대표 문제
지도 학습 정답 라벨 분류, 회귀
비지도 학습 데이터 구조 클러스터링, 차원 축소
강화 학습 보상 게임, 로봇, 정책 최적화

지도 학습¶

입력과 정답 쌍을 이용해 학습합니다. 예: 이미지와 라벨, 집 정보와 가격.

비지도 학습¶

정답 없이 데이터 안의 구조를 찾습니다. 예: 고객 군집화, PCA.

강화 학습¶

행동의 결과로 받은 보상을 최대화하도록 정책을 학습합니다.
