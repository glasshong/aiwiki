앙상블¶

앙상블은 여러 모델의 예측을 결합해 더 안정적인 성능을 얻는 방법입니다.

Bagging¶

Bagging은 여러 모델을 병렬로 학습하고 평균 또는 투표로 예측을 합칩니다. 단일 모델의 variance가 큰 문제를 줄입니다.

Boosting¶

Boosting은 이전 모델이 틀린 샘플에 더 집중하며 순차적으로 학습합니다. 단순 모델의 bias를 낮춰 더 강한 예측기를 만듭니다.

Stacking¶

Stacking은 여러 모델의 예측을 다시 메타 모델의 입력으로 사용합니다. 모델마다 잘 잡는 패턴이 다를 때 유용합니다.
