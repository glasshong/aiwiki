Image Classification¶

Image ClassificationмқҖ мқҙлҜём§Җ м „мІҙм—җ лҢҖн•ҙ н•ҳлӮҳмқҳ лқјлІЁ лҳҗлҠ” м—¬лҹ¬ лқјлІЁмқ„ мҳҲмёЎн•ҳлҠ” taskмһ…лӢҲлӢӨ.

image -> feature extractor -> classifier head -> class probability

м¶ңл Ҙ нҳ•нғң¶

лӢЁмқј лқјлІЁ л¶„лҘҳ:

кі м–‘мқҙ мқҙлҜём§Җ -> cat

лӢӨмӨ‘ лқјлІЁ л¶„лҘҳ:

мӢӨлӮҙ мӮ¬м§„ -> chair, table, window

кё°ліё кө¬мЎ°¶

CNN кё°л°ҳ кө¬мЎ°¶

Input image
-> convolution blocks
-> global average pooling
-> fully connected layer
-> logits

CNNмқҖ мһ‘мқҖ filterлЎң м§Җм—ӯ нҢЁн„ҙмқ„ лҪ‘кі , layerк°Җ к№Ҡм–ҙм§ҲмҲҳлЎқ edge, texture, part, object-level featureлҘј н•ҷмҠөн•©лӢҲлӢӨ.

Transformer кё°л°ҳ кө¬мЎ°¶

Input image
-> patch embedding
-> transformer encoder
-> class token or pooled feature
-> classifier head

ViTлҠ” мқҙлҜём§ҖлҘј patchлЎң лӮҳлҲ„кі  к°Ғ patchлҘј tokenмІҳлҹј лӢӨлЈ№лӢҲлӢӨ.

мҳҲмӢң лӘЁлҚё¶

лӘЁлҚё кө¬мЎ° нҠ№м§•
AlexNet CNN мҙҲкё° л”Ҙлҹ¬лӢқ мқҙлҜём§Җ л¶„лҘҳ лҢҖн‘ң лӘЁлҚё
VGG CNN лӢЁмҲңн•ң 3x3 convolutionмқ„ к№ҠкІҢ мҢ“мқҢ
ResNet CNN + residual skip connectionмңјлЎң к№ҠмқҖ лӘЁлҚё н•ҷмҠө м•Ҳм •нҷ”

LossмҷҖ Metric¶

лӢЁмқј лқјлІЁ л¶„лҘҳлҠ” ліҙнҶө CrossEntropyLossлҘј мӮ¬мҡ©н•©лӢҲлӢӨ.

import torch
import torch.nn as nn

logits = torch.randn(8, 10)   # [batch, num_classes]
targets = torch.randint(0, 10, (8,))

loss = nn.CrossEntropyLoss()(logits, targets)

мһҗмЈј ліҙлҠ” metric:

  • accuracy
  • top-k accuracy
  • precision / recall / F1-score
  • confusion matrix
