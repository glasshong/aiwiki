# 손계산 루틴

AI by Hand의 접근처럼, 모델을 코드로 바로 보기 전에 작은 숫자 예제로 직접 계산하면 구조가 잘 보입니다.

## 추천 순서

1. Black box: 입력과 출력만 봅니다.
2. Components: input, weight, bias, activation, output으로 나눕니다.
3. Tensors: 각 값의 shape을 적습니다.
4. Math: dot product, activation, loss를 수식으로 씁니다.
5. Spreadsheet: 작은 숫자로 검산합니다.
6. Coding: NumPy 또는 PyTorch 결과와 비교합니다.
