# Vision Tasks

## Classification

이미지 전체에 대해 하나의 라벨을 예측합니다.

## Detection

이미지 안의 객체 위치와 클래스를 예측합니다.

## Segmentation

픽셀 단위로 영역을 나눕니다.

| Task | 출력 | 예시 |
|---|---|---|
| Classification | 이미지 라벨 | 고양이 / 강아지 |
| Detection | Bounding box + class | 사람 위치 찾기 |
| Segmentation | Pixel mask | 도로, 차선, 사람 영역 |
