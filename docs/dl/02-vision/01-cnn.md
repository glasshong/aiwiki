# CNN

CNN은 작은 필터를 이미지 위로 이동시키며 엣지, 질감, 모양 같은 지역 특징을 뽑습니다.

## 주요 개념

- Convolution: 필터로 지역 패턴 추출
- Padding: 경계 정보 손실 완화 또는 출력 크기 유지
- Pooling: 공간 크기 축소와 위치 변화에 대한 안정성

## 문제 상황과 해결책

### 경계 정보 손실

Padding은 이미지 주변에 값을 덧대어 경계 영역도 필터가 충분히 볼 수 있게 합니다.

### 위치 변화 민감성

Pooling은 작은 영역의 대표값을 뽑아 작은 위치 변화에 더 안정적인 표현을 만듭니다.
