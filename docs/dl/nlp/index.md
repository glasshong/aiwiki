# NLP

NLP는 텍스트와 언어를 다루는 DL 분야입니다. 현대 NLP는 Transformer와 Attention을 중심으로 발전했습니다.

## 핵심 주제

- RNN / LSTM / GRU
- Transformer
- Attention / Self-Attention / Multi-head Attention
- GPT / BERT
- Zero-shot / Few-shot
- RLHF
- CoT
- Positional Encoding / RoPE
- MoE

## 큰 흐름

문장은 token으로 바뀌고, token embedding은 여러 Layer를 거치며 문맥 정보를 반영합니다.
