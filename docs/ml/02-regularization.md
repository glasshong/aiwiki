# L1 / L2 정규화

정규화는 모델이 학습 데이터에 과하게 맞춰지는 것을 줄이기 위해 가중치 크기에 패널티를 주는 방법입니다.

## L1

L1 정규화는 가중치 절댓값의 합에 패널티를 줍니다. 중요하지 않은 feature의 가중치가 0에 가까워지기 쉬워 특징 선택 효과가 있습니다.

## L2

L2 정규화는 가중치 제곱합에 패널티를 줍니다. 큰 가중치에 더 강한 패널티를 주어 모델이 특정 feature에 과하게 민감해지는 것을 줄입니다.

## 비교

| 구분 | L1 | L2 |
|---|---|---|
| 효과 | 희소성 | 안정화 |
| 가중치 | 일부가 0에 가까워짐 | 전체적으로 작아짐 |
| 사용 느낌 | 특징 선택 | 일반적 과적합 완화 |

## Weight decay

딥러닝에서 자주 말하는 weight decay는 가중치를 조금씩 줄이는 방식의 정규화입니다. AdamW는 gradient 업데이트와 weight decay를 분리해 더 안정적으로 적용합니다.
