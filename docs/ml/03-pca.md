# PCA

PCA는 데이터의 분산을 가장 잘 보존하는 방향을 찾아 차원을 줄이는 비지도 학습 기법입니다.

## 왜 쓰는가

- 고차원 데이터를 낮은 차원으로 압축
- 시각화
- 노이즈 감소
- 모델 입력 특징 수 축소

## 문제 상황과 해결책

### Feature scale 문제

PCA는 분산이 큰 방향을 중요하게 봅니다. feature scale이 다르면 단위가 큰 feature가 과하게 중요해질 수 있습니다.

해결책은 PCA 전에 표준화를 적용하는 것입니다. 각 feature를 평균 0, 표준편차 1에 가깝게 바꾸면 단위 차이가 결과를 지배하는 문제를 줄일 수 있습니다.

### 정보 손실

차원을 너무 많이 줄이면 중요한 정보가 사라질 수 있습니다. explained variance ratio를 확인해 선택한 주성분이 전체 분산의 몇 퍼센트를 보존하는지 봅니다.
