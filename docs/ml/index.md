# ML

Machine Learning은 데이터에서 규칙을 학습해 예측이나 의사결정을 수행하는 방법입니다. 딥러닝보다 더 넓은 개념이며, 전통적인 통계 모델과 트리 모델, 앙상블, 평가 지표가 포함됩니다.

## 먼저 잡을 질문

- 정답 라벨이 있는가?
- 예측하려는 값은 범주인가, 숫자인가?
- 클래스 불균형이 있는가?
- 모델의 정확도보다 중요한 비용이 있는가?

## DL과의 관계

DL은 ML의 하위 분야로 볼 수 있습니다. ML이 사람이 설계한 특징과 비교적 작은 모델을 자주 쓴다면, DL은 신경망이 데이터에서 표현 자체를 학습합니다.
