# Common Training

Common Training은 머신러닝과 딥러닝 모두에 공통으로 등장하는 학습 개념을 모아둔 섹션입니다.

## 왜 분리하나

Loss, gradient, learning rate, optimizer는 딥러닝에서 자주 보이지만 개념 자체는 모델을 학습시키는 거의 모든 방법에 연결됩니다.

```text
데이터 -> 예측 -> Loss 계산 -> Gradient 계산 -> Optimizer 업데이트 -> 평가
```
