# AI Knowledge Map

AI 지식을 `Common Training`, `ML`, `DL`로 나눠 정리한 문서 사이트입니다.

## 큰 구조

### Common Training

머신러닝과 딥러닝 학습에 공통으로 적용되는 개념을 다룹니다.

- Loss Function
- Gradient & Backpropagation
- Learning Rate
- Optimizer
- Hyperparameter Tuning
- Forward / Backward

### ML

전통 머신러닝과 평가 지식을 다룹니다.

- 학습 유형
- L1 / L2 정규화
- PCA
- 앙상블
- 평가 지표

### DL

신경망 기반 학습과 현대 모델을 다룹니다.

- Train: Activation, Normalization, Dropout, Batch Size, MLP/FCL, Tensor/Channel
- Vision: CNN, Detection, Segmentation, ViT, Swin Transformer, DETR
- NLP: RNN, Transformer, Attention, GPT/BERT, RLHF, CoT, Autoencoder, MoE
- Optimization: Quantization, Pruning, LoRA/QLoRA

## 문서 작성 원칙

각 주제는 정의, 왜 중요한지, 핵심 개념, 문제 상황과 해결책, 적용 예시 순서로 정리합니다.
