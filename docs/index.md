# AI Knowledge Map

AI 지식을 `ML`과 `DL`로 나눠 정리한 문서형 웹사이트입니다.

## 큰 구조

### ML

전통 머신러닝과 평가 지표를 다룹니다.

- 학습 유형
- L1 / L2 정규화
- PCA
- 앙상블
- 평가 지표

### DL

신경망 기반 학습과 도메인 모델을 다룹니다.

- Train: Loss, Gradient, Optimizer, Learning Rate, Activation
- Vision: CNN, Detection, Segmentation, ViT
- NLP: RNN, Transformer, Attention, GPT/BERT
- Optimization: Quantization, Pruning, LoRA/QLoRA

## 문서 작성 원칙

각 주제는 정의, 왜 중요한지, 핵심 개념, 문제 상황과 해결책, 작은 예시 순서로 정리합니다.
